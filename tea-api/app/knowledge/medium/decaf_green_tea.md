# 商品名：カフェインカット（デカフェ）緑茶

## 特徴・説明
- **特徴**: 深蒸し茶のコクそのままカフェイン70％カットの「家族で楽しむデカフェ緑茶」
- **説明**: 「妊娠中も、夜も、子どもと一緒でも。一番に選んでほしい安心感」 深いコクと鮮やかな色が特徴の深蒸し緑茶から、風味はそのままにカフェインだけを70％カットしました。カフェインは控えたいけれど、お茶本来の美味しさは妥協したくない。そんな願いに寄り添う、からだ想いの緑茶です。
  - 深蒸し茶の豊かな味わい 独自の製法でカフェインを70％除去。デカフェ特有の物足りなさを感じさせない、深蒸し茶らしいまろやかな旨みと香りが広がります。
  - 水と二酸化炭素だけで抽出 化学薬品（有機溶媒）を一切使わない「超臨界二酸化炭素抽出法」を採用。妊婦さんや小さなお子様も、毎日安心してお飲みいただけます。
  - 自然に体にも優しい植物由来のティーバッグ プラスチック不使用の環境と体にやさしい素材を使用。お湯に浸しても雑味がなく、「やさしさ」へのこだわりを徹底しました。
- **シーン**: 
  - 妊娠中・授乳中のホッとしたいひとときに。
  - おやすみ前のリラックスタイムに。
   - お子様と一緒に囲む食卓の飲み物として。
- **原材料名**: 日本茶（深蒸し緑茶）
- **原産地**: 三重県
- **内容**: ティーバッグ8個
- **賞味期限**: 製造から1年
