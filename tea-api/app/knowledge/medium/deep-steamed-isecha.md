# 商品名：伊勢の深蒸し茶

## 特徴・説明
- **特徴**: 深い緑、満たされるコク。 伊勢の大地が育んだ、贅沢な「深蒸し」のひととき
- **説明**: 日本有数の茶産地三重県。その豊かな風土で育った茶葉を熟練の技術で通常の倍以上の時間をかけて丁寧に蒸し上げたのがこの「伊勢の深蒸し茶」です。
  - 深蒸し茶だけの「とろり」とした濃厚さ：長く蒸すことで茶葉が細かくなり、急須で淹れた際に茶葉そのものの成分がたっぷりと溶け出します。見た目にも鮮やかな濃い緑色と、とろりとした濃厚な旨み、そして渋みの少ないまろやかな後味が特徴です。
  - 心解き放つ、くつろぎの香り：お湯を注いだ瞬間に立ち昇る、若草のような瑞々しく芳醇な香り。忙しい日常の手を止めて、五感で味わう本格的な日本茶の時間は、至福のリラックスタイムを演出します。
  - 贈答用にも選ばれる品質：ご自宅用としてはもちろん、その確かな品質と味わいは、お茶好きの方への贈り物や特別なお客様をお迎えする際の一杯としても、自信を持っておすすめいたします。
- **原材料名**: 日本茶（深蒸し緑茶）
- **原産地**: 三重県
- **内容**: ティーバッグ10個
- **賞味期限**: 製造から1年

## おすすめの淹れ方・レシピ
1. **湯量**: 200cc
2. **湯温**: 70℃〜80℃
3. **抽出時間**: 90〜120秒
4. **淹れ方**: 深蒸し茶は茶葉が細かいため、網目の細かい急須（深蒸し急須）をお使いいただくと、より一層美味しく召し上がれます。少し低めのお湯（約70℃〜80℃）でゆっくりと抽出することで、甘みがさらに引き立ちます。待っている間に、紐を引っ張ってちょんちょんしてもOK。
