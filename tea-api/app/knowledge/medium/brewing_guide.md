# 美味しい伊勢茶の淹れ方ガイド

藤八茶寮の伊勢茶をより美味しくお楽しみいただくための基本レシピです。シングルオリジンならではの力強い味わいと香りを引き出すポイントをまとめました。

## 1. 深蒸し茶（ふかむしちゃ）
葉が細かく、濃厚なコクと鮮やかな緑色が特徴です。
* **茶葉の量**: 4g（小さじ2杯弱）
* **お湯の量**: 200ml
* **お湯の温度**: 70℃〜80℃（沸騰したお湯を湯呑みに移し、一呼吸おいたくらいが目安です）
* **抽出時間**: 30秒〜45秒
* **ポイント**: 最後の一滴まで絞りきることで、旨みが凝縮された濃厚な味わいになります。

## 2. ほうじ茶
香ばしい香りと、渋みのないすっきりとした甘みが楽しめます。
* **茶葉の量**: 3g〜4g（大さじ2杯ほど、嵩があるため多めに）
* **お湯の量**: 200ml
* **お湯の温度**: 90℃〜100℃（熱湯で淹れることで香りが一気に立ち上がります）
* **抽出時間**: 30秒
* **ポイント**: 立ち上る「香ばしさ」を楽しむお茶ですので、ぜひ熱々のお湯で淹れてください。

## 3. 和紅茶（わこうちゃ）
三重県産茶葉で作られた和紅茶は、渋みが少なく、自然でやさしい甘みが特徴です。
* **茶葉の量**: 3g（小さじ1杯強）
* **お湯の量**: 200ml
* **お湯の温度**: 90℃〜100℃
* **抽出時間**: 2分〜3分（じっくり蒸らしてください）
* **ポイント**: 砂糖を入れなくても甘みを感じられるため、まずはストレートで飲むのがおすすめです。

## 4. 伊勢茶パウダー
お料理やスイーツ作り、ラテにも手軽に使える粉末タイプです。
* **基本の飲み方**: 小さじ1/2をお湯または水（150ml）でよく混ぜるだけ。
* **ほうじ茶ラテ**: パウダーを少量のお湯で練り、温めた牛乳と砂糖を加えるとお店のような味わいになります。
