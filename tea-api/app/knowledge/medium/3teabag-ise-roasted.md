# 商品名：【ティーバッグ3個】伊勢のほうじ茶

## 特徴・説明
- **特徴**: いつでも、どこでも、至福の香りを。三重県産100%「伊勢のほうじ茶」ティーバッグ
- **説明**: 日本有数のお茶処・三重県で育まれた「伊勢茶」の中でも、特に香りの良い「一番茶」のみを贅沢に使用しました。 熟練の焙煎技術でじっくりと強火で焙じ、封を開けた瞬間に広がる香ばしさは、心まで解きほぐす深い癒やしを届けます。
  - 一番茶ならではの深い旨み: 一般的なほうじ茶に使用される三番茶や四番茶ではなく、一番茶の茶葉や茎を使用。苦みや渋みが少なく、お茶本来の甘みが際立ちます。
  - 水出しでも熱湯でも：茎だけを強く焙じることで焙じ香と茎の甘みを引き出し、すっきりと仕上げていますので水出しにするとゴクゴク行ける一方熱湯で淹れることで香りが爆発的にたちます。
  - 低カフェインで体に優しい: 焙煎によってカフェインが抑えられているため、就寝前のリラックスタイムや、お子様、妊娠中・授乳中の方でも安心してお楽しみいただけます。
- **原材料名**: 日本茶（ほうじ茶）
- **原産地**: 三重県
