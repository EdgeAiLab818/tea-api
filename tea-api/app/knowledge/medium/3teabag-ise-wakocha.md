# 商品名：【ティーバッグ3個】伊勢の和紅茶

## 特徴・説明
- **特徴**: 日本の風土が育んだ、優しく甘い香り。三重県産100%「伊勢の和紅茶」
- **説明**: 和紅茶：日本で育った茶葉を使った国産紅茶です。海外の紅茶に比べて渋みが少なく、やさしくすっきりとした後味が特徴です。緑茶と紅茶の間のような風味をぜひお楽しみください
  - ストレートで美味しい:渋みが少ないため、まずはぜひお砂糖を入れずにストレートで。伊勢茶ならではの、まろやかで奥深い香りをお楽しみいただけます
  - 和食や和菓子にも合う: 日本の茶葉（緑茶）から作られているため、洋菓子はもちろん、和菓子や普段のお食事との相性も抜群です
- **原材料名**: 日本茶（和紅茶）
- **原産地**: 三重県
- **内容**: ティーバッグ3個
- **賞味期限**: 製造から1年
- **淹れ方**: 急須がなくても、カップ一杯から本格的な和紅茶を楽しめるテトラ型ティーバッグを採用。茶葉がしっかり躍り、香りを引き出します
