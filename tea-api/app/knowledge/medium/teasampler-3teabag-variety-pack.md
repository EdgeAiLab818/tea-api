# 商品名：【飲み比べ】ティーバッグ３種セット

## 特徴・説明
- **特徴**: 伊勢茶の魅力を、この一包に。人気の3種を一度に楽しむ「贅沢飲み比べセット」
- **説明**: 「藤八茶寮」自慢のシングルオリジン伊勢茶を、手軽なティーバッグで詰め合わせました。 濃厚なコクの「深蒸し茶」、香ばしい「ほうじ茶」、そして希少な「和紅茶」。それぞれに個性豊かな三重県産100%の味わいを、その日の気分に合わせてお楽しみいただけます。
  - 伊勢の深蒸し茶: 通常の2倍以上の時間をかけて蒸し上げた、濃厚なコクと鮮やかな緑が特徴の本格煎茶。
  - 伊勢のほうじ茶: 一番茶を贅沢に焙じた、芳醇な香りとまろやかな甘みが広がる心休まる一杯。
  - 伊勢の和紅茶: 渋みが少なく、お砂糖なしでもほのかな甘みを感じる、優しく上品な日本の紅茶。
