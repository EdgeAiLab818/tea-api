# 商品名：伊勢ほうじ茶パウダー（無糖）

## 特徴・説明
- **特徴**: 伊勢の香ばしさを丸ごと味わう。無添加・無着色の「伊勢のほうじ茶パウダー」
- **説明**: 三重県産伊勢茶の「一番茶」を贅沢に使用し、丁寧に焙じ上げたほうじ茶を細かな粉末に仕上げました。 茶葉をそのまま粉砕しているため、お茶の栄養成分を丸ごと摂取でき、ほうじ茶本来の圧倒的な香ばしさをダイレクトに楽しめます。
  - ほうじ茶ラテに: 牛乳や豆乳に溶かすだけで、カフェのような本格ほうじ茶ラテが完成。お好みで蜂蜜などを加えてもOK。
  - お菓子作り・製菓用に: ほうじ茶スイーツ（プリン、シフォンケーキ、クッキー）の材料として。熱を加えても香りが飛びにくいのが特徴です。
  - お料理のアクセントに: アイスクリームのトッピングや、ほうじ茶塩として天ぷらなどにも。
  - 健康習慣に: カフェイン控えめで、お休み前のリラックスタイムにも最適。無糖なので糖質を気にされる方にも安心です。
- **原材料名**: 日本茶（ほうじ茶粉茶）
- **原産地**: 三重県
- **内容**: 80g
- **賞味期限**: 製造から1年

## おすすめの淹れ方・レシピ
1. **飲み方**: ほうじ茶ラテ
2. **材料**: ほうじ茶パウダー　3g。おさとう　小さじ1杯（おこのみで！）。牛乳 160cc
3. **湯量**: 25~30ml
4. **淹れ方**: 粉末ほうじ茶をよくお茹で溶かし、温めた牛乳を注いで完成◎
5. **その他のレシピ**: クッキー、パウンドケーキ、アイスクリームなど
