# 商品名：伊勢のほうじ茶

## 特徴・説明
- **特徴**: 幕末より続く伝統。三重県産100%「伊勢のほうじ茶」で心安らぐひとときを
- **説明**: 
  - 日本有数のお茶処、三重県で育まれた「伊勢茶」を100%使用したこだわりのほうじ茶です。 「伊勢のほうじ茶」は、厳選された一番茶の茶葉や茎を熟練の技術でじっくりと強火で焙じました。
  - 芳醇な香りと甘み: 封を開けた瞬間に広がる香ばしさと、口の中に広がるお茶本来のほのかな甘みが特徴です。

  - 低カフェインで安心: ほうじ茶はカフェインが少ないため、お子様や妊婦の方、おやすみ前のリラックスタイムにも安心してお召し上がりいただけます。

## おすすめの淹れ方・レシピ
1. **湯量**: 200cc
2. **湯温**: 90℃
3. **材料**: 茶葉3gまたはティーバッグ1個
4. **抽出時間**: 90〜120秒
5. **淹れ方**:湯呑みに茶葉3g（またはティーバッグ1個）を入れる。200ccほどお湯を注ぐ。90〜120秒ほど待つ。待っている間に、紐を引っ張ってちょんちょんしてもOK。ホットは同じ茶葉にお湯を足すことで2煎目、3煎目も楽しめます。
